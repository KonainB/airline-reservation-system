# airline-reservation-system
### By Konain
#### Introduction
_This project provides functionality to view flights using CRUD functionality._

### Features Included:
#### 
*Flight destination

*Flight date and time

*Create new customer

*Login page

*Read customer details

*Update customer details

*Delete customer details

*Create flights

*Read flights

*Update flights

*Delete flights
